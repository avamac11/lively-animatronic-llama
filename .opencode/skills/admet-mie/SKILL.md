# ADMET-MIE Skill

## What is ADMET-MIE?

ADMET-MIE (Absorption, Distribution, Metabolism, Excretion, Toxicity - Molecular Initiating Event) is a skill that bridges the gap between ADMET property predictions and adverse outcome pathway analysis. It identifies potential Molecular Initiating Events (MIEs) that could trigger adverse outcomes based on a molecule's chemical structure and predicted ADMET properties.

## Overview
This skill integrates ADMET property predictions with molecular initiating event analysis to support Adverse Outcome Pathway (AOP) prediction. It uses chemical structure analysis, functional group identification, and ADMET property patterns to predict potential MIEs that could lead to toxicological outcomes. The skill uses results of chemical descriptors and analysis generated by the ADMET-AI skill to predict potential MIEs that could lead to toxicological outcomes. The MIEs are predicted using a combination of chemical structure analysis, functional group identification, and ADMET property patterns. The skill also integrates with the AOP Wiki to retrieve known MIEs and their associated AOPs for comparison and validation.


## Key Features

### Molecular Initiating Event Prediction
- **Structure-based MIE identification**: Analyzes chemical functional groups associated with known MIEs
- **ADMET-driven MIE prediction**: Uses ADMET property patterns to identify potential toxicological mechanisms


## Implementation
The skill uses RDKit for chemical structure analysis and integrates with the existing ADMET scoring capabilities to predict potential MIEs. It is for use with/after the ADMET-AI skill, which provides ADMET property predictions.

### Architecture
The ADMET-MIE skill is implemented as a Python class with the following structure:

```
ADMETMIESkill
├── __init__() - Initialize the skill with AOP Wiki client
├── predict_molecular_initiating_events() - Predict MIEs from SMILES
├── admet_to_mie_mapping() - Map ADMET properties to MIEs
├── assess_toxicity_risk() - Assess overall toxicity risk
├── get_aop_predictions() - Generate comprehensive AOP predictions
├── get_known_miess_from_aop_wiki() - Retrieve MIEs from AOP Wiki
├── search_miess() - Search MIEs in AOP Wiki
├── get_aop_data_for_mie() - Get AOP data for specific MIE
└── Context manager support (__enter__, __exit__)
```

### AOP Wiki API Integration
The skill integrates with the AOP Wiki API through the `AOPWikiClient` class, which provides:

- **`get_miess_for_chemical()`**: Retrieve MIEs for specific chemicals by name or CAS RN
- **`search_miess()`**: Search for MIEs matching query strings
- **`get_all_aops()`**: Retrieve all Adverse Outcome Pathways
- **`get_aop_by_id()`**: Get specific AOP by ID
- **`_parse_aop()`**: Parse AOP data into structured objects

The integration follows these patterns:
1. **Direct API calls**: For specific chemical lookups and searches
2. **Batched processing**: For retrieving all AOPs and extracting MIEs
3. **Error handling**: Graceful handling of API failures and timeouts
4. **Context management**: Proper session cleanup

### Core Functions
1. **`predict_molecular_initiating_events(smiles: str) -> dict`**: Predicts potential MIEs for a given molecule
2. **`assess_toxicity_risk(smiles: str) -> dict`**: Assesses overall toxicity risk based on MIE predictions
3. **`get_known_miess_from_aop_wiki(chemical_name: str = None, cas_rn: str = None) -> dict`**: Retrieves known MIEs from AOP Wiki for a specific chemical
4. **`get_aop_data_for_mie(mie_id: int) -> dict`**: Retrieves AOP data associated with a specific MIE
5. **`search_miess(query: str) -> dict`**: Searches for MIEs matching a query string

### Function Details

#### `predict_molecular_initiating_events(smiles: str) -> dict`
Predicts potential Molecular Initiating Events for a given molecule based on chemical structure analysis.

**Parameters:**
- `smiles` (str): SMILES representation of the molecule

**Returns:**
- `dict`: Dictionary containing predicted MIEs with confidence scores and descriptions

**Example:**
```python
result = admet_mie.predict_molecular_initiating_events("c1ccccc1")
# Returns: {
#   "predicted_miess": [
#       {"mie_name": "Covalent binding to protein", "confidence": 0.75, "description": "..."},
#       {"mie_name": "Oxidative stress", "confidence": 0.60, "description": "..."}
#   ],
#   "smiles": "c1ccccc1"
# }
```

#### `get_known_miess_from_aop_wiki(chemical_name: str = None, cas_rn: str = None) -> dict`
Retrieves known Molecular Initiating Events from AOP Wiki for a specific chemical.

**Parameters:**
- `chemical_name` (str, optional): Name of the chemical to search for
- `cas_rn` (str, optional): CAS registry number of the chemical

**Returns:**
- `dict`: Dictionary containing known MIEs with detailed information

**Example:**
```python
result = admet_mie.get_known_miess_from_aop_wiki(chemical_name="benzene")
# Returns: {
#   "count": 3,
#   "miess": [
#       {
#           "id": 123,
#           "name": "Covalent binding to protein",
#           "chemical_name": "benzene",
#           "cas_rn": "71-43-2",
#           "smiles": "c1ccccc1",
#           "target": "Nrf2 pathway",
#           "evidence": "...",
#           "references": [...]
#       }
#   ]
# }
```

#### `search_miess(query: str) -> dict`
Searches for Molecular Initiating Events matching a query string in the AOP Wiki.

**Parameters:**
- `query` (str): Search query string

**Returns:**
- `dict`: Dictionary containing search results with MIEs matching the query

**Example:**
```python
result = admet_mie.search_miess(query="oxidative stress")
# Returns: {
#   "count": 15,
#   "miess": [
#       {"id": 456, "name": "Oxidative stress via Nrf2 pathway", ...},
#       {"id": 789, "name": "Mitochondrial oxidative stress", ...}
#   ]
# }
```

#### `get_aop_data_for_mie(mie_id: int) -> dict`
Retrieves full Adverse Outcome Pathway data associated with a specific Molecular Initiating Event.

**Parameters:**
- `mie_id` (int): ID of the MIE to retrieve data for

**Returns:**
- `dict`: Dictionary containing AOP data for the specified MIE

**Example:**
```python
result = admet_mie.get_aop_data_for_mie(123)
# Returns: {
#   "mie_id": 123,
#   "mie_name": "Covalent binding to protein",
#   "aop_id": 1,
#   "aop_title": "Benzene-induced hematotoxicity",
#   "aop_description": "...",
#   "mie_details": {...},
#   "key_events": [{"id": 1, "name": "Molecular initiating event", ...}, ...]
# }
```

#### `assess_toxicity_risk(smiles: str) -> dict`
Assesses overall toxicity risk based on MIE predictions and provides risk levels.

**Parameters:**
- `smiles` (str): SMILES representation of the molecule

**Returns:**
- `dict`: Dictionary containing toxicity risk assessment with risk score and level

**Example:**
```python
result = admet_mie.assess_toxicity_risk("c1ccccc1")
# Returns: {
#   "smiles": "c1ccccc1",
#   "risk_score": 0.72,
#   "risk_level": "High",
#   "high_risk_miess": ["Covalent binding to protein"],
#   "predicted_miess": [...]
# }
```

#### `get_aop_predictions(smiles: str) -> dict`
Generates comprehensive Adverse Outcome Pathway predictions based on MIE analysis.

**Parameters:**
- `smiles` (str): SMILES representation of the molecule

**Returns:**
- `dict`: Dictionary containing comprehensive AOP predictions

**Example:**
```python
result = admet_mie.get_aop_predictions("c1ccccc1")
# Returns: {
#   "smiles": "c1ccccc1",
#   "toxicity_assessment": {...},
#   "known_miess_from_aop_wiki": {...},
#   "recommended_follow_up": ["Conduct in vitro toxicity testing", ...]
# }
```

### Example Usage

```python
from admet_mie import ADMETMIESkill
from aop_wiki_api import AOPWikiClient

# Initialize the ADMET-MIE skill
admet_mie = ADMETMIESkill()

# Example 1: Predict Molecular Initiating Events from SMILES
smiles = "c1ccccc1"  # Benzene
prediction = admet_mie.predict_molecular_initiating_events(smiles)
print(f"Predicted MIEs for {smiles}: {prediction}")

# Example 2: Get known MIEs from AOP Wiki for a chemical
chemical_name = "benzene"
known_miess = admet_mie.get_known_miess_from_aop_wiki(chemical_name=chemical_name)
print(f"Known MIEs for {chemical_name}: {known_miess}")

# Example 3: Search for MIEs matching a query
query = "oxidative stress"
search_results = admet_mie.search_miess(query=query)
print(f"Search results for '{query}': {search_results}")

# Example 4: Get AOP data for a specific MIE
mie_id = 123  # Example MIE ID
aop_data = admet_mie.get_aop_data_for_mie(mie_id)
print(f"AOP data for MIE {mie_id}: {aop_data}")

# Example 5: Comprehensive toxicity assessment and AOP prediction
smiles = "c1ccccc1"
comprehensive_result = admet_mie.get_aop_predictions(smiles)
print(f"Comprehensive AOP prediction for {smiles}:")
print(f"  Risk level: {comprehensive_result['toxicity_assessment']['risk_level']}")
print(f"  Recommended follow-up: {comprehensive_result['recommended_follow_up']}")

# Example 6: Direct AOP Wiki API usage
with AOPWikiClient() as client:
    # Get MIEs for a specific chemical
    miess = client.get_miess_for_chemical(chemical_name="benzene")
    for mie in miess:
        print(f"MIE: {mie.name}")
        if mie.smiles:
            print(f"  SMILES: {mie.smiles}")
        if mie.cas_rn:
            print(f"  CAS RN: {mie.cas_rn}")
    
    # Get all AOPs
    all_aops = client.get_all_aops()
    print(f"Found {len(all_aops)} AOPs")
    
    # Get a specific AOP by ID
    aop = client.get_aop_by_id(1)
    if aop:
        print(f"AOP {aop.id}: {aop.title}")
        print(f"MIEs in this AOP: {len(aop.molecular_initiating_events)}")

# Example 7: Context manager usage
with ADMETMIESkill() as admet_mie:
    result = admet_mie.assess_toxicity_risk("c1ccccc1")
    print(f"Toxicity risk assessment: {result}")
```

- **Compression**: Enable response compression when available

**Caching Strategies:**
```python
# Simple in-memory cache example
class CachedADMETMIESkill(ADMETMIESkill):
    def __init__(self):
        super().__init__()
        self._mie_cache = {}
        self._aop_cache = {}
    
    def get_known_miess_from_aop_wiki(self, chemical_name: str = None, cas_rn: str = None) -> Dict:
        cache_key = f"{chemical_name}_{cas_rn}"
        if cache_key in self._mie_cache:
            return self._mie_cache[cache_key]
        
        result = super().get_known_miess_from_aop_wiki(chemical_name, cas_rn)
        if "error" not in result:
            self._mie_cache[cache_key] = result
        
        return result
```

**Performance Metrics:**
- **API response times**: Typically 1-5 seconds for individual requests
- **Batch processing**: Retrieving all AOPs may take 30+ seconds
- **Memory footprint**: AOP data can range from MBs to GBs depending on dataset size
- **Network usage**: API responses can be several MBs for comprehensive data

## AOP Wiki Data Models

The AOP Wiki API integration uses structured data models to represent AOP information:

### MolecularInitiatingEvent
Represents a Molecular Initiating Event with the following attributes:
- `id`: Unique identifier for the MIE
- `name`: Name of the MIE
- `description`: Description of the MIE
- `event_type`: Type of event (always "Molecular Initiating Event")
- `evidence`: Evidence supporting the MIE
- `references`: List of references
- `chemical_name`: Name of the associated chemical
- `cas_rn`: CAS registry number
- `smiles`: SMILES representation
- `inchi`: InChI representation
- `inchi_key`: InChI key
- `target`: Molecular target

### KeyEvent
Represents a general Key Event in an AOP:
- `id`: Unique identifier
- `name`: Event name
- `description`: Event description
- `event_type`: Type of event
- `evidence`: Supporting evidence
- `references`: Reference list

### AOP
Represents an Adverse Outcome Pathway:
- `id`: Unique AOP identifier
- `title`: AOP title
- `description`: AOP description
- `key_events`: List of all key events
- `molecular_initiating_events`: Property that returns only MIEs from key events
