# Data Sources Guide

This guide provides a comprehensive list of databases, web sources, and tools for gathering information related to chemical properties, biological pathways, toxicology, and more. This document can be edited and expanded as new sources are discovered or needed.

## Chemical Properties and Interactions

### PubChem
- **URL**: [https://pubchem.ncbi.nlm.nih.gov/](https://pubchem.ncbi.nlm.nih.gov/)
- **Description**: PubChem is the world's largest collection of freely accessible chemical information.
- **Features**:
  - Search chemicals by name, molecular formula, structure, and other identifiers
  - Access chemical and physical properties, biological activities, safety and toxicity information
  - Browse patents, literature citations, and more
  - Programmatic access via API and downloads
  - Over 1,120 data sources, 123M compounds, 347M substances, 299M bioactivities
- **Use Cases**:
  - Finding chemical properties and structures
  - Exploring biological activities and safety information
  - Accessing literature and patent data

### ChemSpider
- **URL**: [http://www.chemspider.com/](http://www.chemspider.com/)
- **Description**: ChemSpider is a free chemical structure database providing fast access to over 100 million structures, properties, and associated information.
- **Features**:
  - Chemical structure search
  - Property prediction
  - Integration with other databases
- **Use Cases**:
  - Chemical structure validation
  - Property prediction
  - Cross-referencing with other databases

### ChEMBL
- **URL**: [https://www.ebi.ac.uk/chembl/](https://www.ebi.ac.uk/chembl/)
- **Description**: ChEMBL is a manually curated database of bioactive molecules with drug-like properties.
- **Features**:
  - Bioactivity data
  - Drug targets
  - Compound information
- **Use Cases**:
  - Drug discovery research
  - Bioactivity data analysis
  - Target identification

## Chemical-Gene Interactions and Pathways

### Comparative Toxicogenomics Database (CTD)
- **URL**: [https://ctdbase.org/](https://ctdbase.org/)
- **Description**: CTD is a public database that curates data to illustrate the relationships between chemicals, genes, and diseases.
- **Features**:
  - Chemical-gene interactions
  - Chemical-disease relationships
  - Gene-disease associations
- **Use Cases**:
  - Understanding chemical impacts on biological systems
  - Disease mechanism research
  - Toxicogenomics studies

### STITCH
- **URL**: [https://stitch.embl.de/](https://stitch.embl.de/)
- **Description**: STITCH is a database of known and predicted interactions between chemicals and proteins.
- **Features**:
  - Chemical-protein interactions
  - Interaction networks
  - Confidence scores
- **Use Cases**:
  - Building interaction networks
  - Predicting protein targets
  - Understanding chemical mechanisms

### DrugBank
- **URL**: [https://go.drugbank.com/](https://go.drugbank.com/)
- **Description**: DrugBank is a comprehensive, freely accessible database containing in-depth information on drugs and their targets.
- **Features**:
  - Drug-target interactions
  - Drug metabolism information
  - Pharmacokinetic data
- **Use Cases**:
  - Drug-target interaction analysis
  - Pharmacokinetic studies
  - Drug development research

## Biological Pathways

### KEGG (Kyoto Encyclopedia of Genes and Genomes)
- **URL**: [https://www.kegg.jp/](https://www.kegg.jp/)
- **Description**: KEGG is a database resource for understanding high-level functions and utilities of biological systems.
- **Features**:
  - Pathway maps
  - BRITE hierarchies
  - KEGG modules
  - KO (KEGG Orthology) functional orthologs
  - Genomes with links to pathways
- **Use Cases**:
  - Pathway analysis
  - Functional annotation
  - Genome analysis

### Reactome
- **URL**: [https://reactome.org/](https://reactome.org/)
- **Description**: Reactome is a free, open-source, curated, and peer-reviewed pathway database.
- **Features**:
  - Pathway browser
  - Analysis tools
  - AI chatbot for pathway queries
  - Human pathways (2,883)
  - Reactions (16,423)
  - Proteins (11,694)
- **Use Cases**:
  - Pathway visualization
  - Functional enrichment analysis
  - Gene set analysis

### WikiPathways
- **URL**: [https://www.wikipathways.org/](https://www.wikipathways.org/)
- **Description**: WikiPathways is an open science platform for biological pathways contributed, updated, and used by the research community.
- **Features**:
  - Community-curated pathways
  - Pathway analysis tools
  - Download options (GPML, GMT, SVG)
  - API and SPARQL access
- **Use Cases**:
  - Pathway browsing and sharing
  - Pathway analysis
  - Data visualization

### Pathway Commons
- **URL**: [https://www.pathwaycommons.org/](https://www.pathwaycommons.org/)
- **Description**: Pathway Commons is a collection of biological pathway data integrated from public pathway databases.
- **Features**:
  - Integrated pathway data
  - Search and visualization tools
  - BioPAX format
- **Use Cases**:
  - Cross-database pathway analysis
  - Network visualization
  - Systems biology research

## Tips for Using These Resources

1. **Cross-Reference Data**: Always cross-reference data from multiple sources to ensure accuracy and completeness.

2. **Use APIs**: Many databases offer APIs for programmatic access, which can be useful for large-scale data retrieval and analysis.

3. **Check Data Updates**: Ensure you are using the most recent data by checking the update frequencies of each database.

4. **Cite Sources**: Always cite the sources of your data in your research to maintain transparency and credibility.

5. **Explore Tools**: Many databases offer integrated tools for analysis and visualization, which can save time and effort.

6. **Community Resources**: Engage with the scientific community through forums, discussions, and social media to get tips and updates on these resources.
